What's this

Provides tools to validate an XML document against a set of XSD schemas in Javascript. Under heavy development, so things might change quickly. Usage examples are coming soon.

Requirements

Defined in a RequireJS format, so you need an AMD loader.

Uses: - Underscore.js - wicked-good-xpath - requirejs-text - Some of my own projects: kapa-xsd, kapa-objTools, kapa-Library, kapa-primiveUnserializers.

Tests

Tests are written in Jasmine 2.0 and I run them with Karma.

Documentation

All the methods are documented with JSDoc3, you can find the generated HTML documentation in the docs folder.
